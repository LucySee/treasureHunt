# 🗺️ Treasure Hunt App

A fun treasure hunt game for kids to play at home! Parents create clue hunts with drawings and secret codes, and kids solve them one by one to find the treasure.

Give it a try at https://lucysee.github.io/treasureHunt/

## 🎮 How It Works

### For Parents
1. Tap **Create Hunt** — give your hunt a name
2. Write a clue and/or draw a hint map for each location
3. Tap **🎲 Auto** to generate a secret code (or type your own)
4. Write the code on a piece of paper and **hide it at the clue location**
5. Tap **+ Add Clue** and repeat for each hiding spot
6. Tap **💾 Save Hunt** when done
7. Use **🔐 Clues & Codes** anytime to check your codes

### For Kids
1. Tap **🎮 Start Hunt**
2. Read or look at the clue drawing
3. Find the hiding spot and look for the secret code
4. Enter the code on the number pad to unlock the next clue
5. Find all the clues to discover the treasure! 🎉

## ✨ Features

- 🎨 Draw clue hints with colours and brush sizes
- 🎲 Auto-generate secret codes
- 📋 Parent cheat sheet with all clues and codes
- 🎉 Exciting celebration when treasure is found
- 💾 Hunts saved locally in your browser
- 📱 Works on phone, tablet and desktop


## 📝 Notes

- Hunts are stored in your browser's local storage
- Each device has its own separate hunt storage
- Clearing browser data will erase saved hunts
