❓ How to Play

🎨 How do I create a hunt?
Tap Create Hunt. Write a clue, draw a hint if you like, tap 🎲 Auto to generate a code, then tap + Add Clue. Repeat for each hiding spot, then Save Hunt.
🔐 What do I do with the secret code?
Write the code on a piece of paper and hide it at the clue location. Kids must physically find the hiding spot to get the code!
🎮 How do kids play?
Tap Start Hunt, pick a hunt, read the clue, find the hiding spot, then enter the secret code on the number pad to unlock the next clue.
💾 Are my hunts saved?
Yes! Hunts are saved on this device and stay until you delete them or clear your browser data.
🔍 I forgot the code — where can I check?
Tap Clues & Codes on the home screen to see all your clues and their secret codes anytime.
🔥 How to make the hunt more challenging?
Instead of writing the code in plain numbers, try these fun ways to hide it!

🔢 Different Language
Write in Malay: satu-dua-tiga-empat = 1234

🏛️ Roman Numerals
I · V · III · II = 1532

🌈 Rainbow Color Code
Each color = a number. Red=1, Orange=2, Yellow=3, Green=4, Blue=5, Purple=6
Draw colored dots 🔴🟡🟢🔵 = 1342

🔤 Alphabet Code
A=1, B=2, C=3... so A · B · D · C = 1243

🎲 Dice Dots
Draw dice faces: ⚀⚃⚁⚄ = 1425

🕐 Clock Time
Draw clock faces showing the hour: 3 o'clock · 7 o'clock · 1 o'clock · 9 o'clock = 3719

➕ Math Puzzle
Write: 1+1 · 4-2 · 1+2 · 2×2 = 2234

Tip: Draw the code clue on paper and hide it at the location!
